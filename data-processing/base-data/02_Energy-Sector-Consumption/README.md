# Energy Sector Consumption Data

Detailed energy consumption by economic sector.

## Datasets

### end_uses_efficiency/
IEA Energy End-uses and Efficiency Indicators Database (December 2025 Highlights).

**Coverage**: 50+ countries, 2000-2023
**Sectors**: Industry, Residential, Services, Transport
**Files**: 22 CSV files (~3.6 MB total)
MAIN are the EXCEL TABLES
WorldEnergyBalancesHighlights2025.xlsx
IEA - Energy End-uses and Efficiency Indicators database - Highlights (December 2025).xlsb

See `end_uses_efficiency/README.md` for detailed documentation.

Key file categories:
- `*_energy.csv` - Energy consumption by end-use and product
- `*_emissions.csv` - CO2 emissions data
- `*_energy_indicators.csv` - Efficiency indicators
- `*_carbon_indicators.csv` - Carbon intensity metrics
- `*_decomposition.csv` - Decomposition analysis

### WorldEnergyBalancesHighlights2025.xlsx
IEA World Energy Balances Highlights (2025 edition).

**Format**: Excel workbook
**Coverage**: Global energy balances
**Note**: Excel file - see IEA documentation for structure
